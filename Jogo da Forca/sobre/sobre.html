Sobre o projecto

Este site surgiu como um projeto final no âmbito da cadeira de ITW - Introdução às Tecnologias Web.

O projeto foi criado com o objetivo de evoluir na criação, contrução e implementação de sites.

O passo inicial para a criação de um site, ou seja para a criação de páginas Web,é o HTML - HyperText Markup Language, é por isso que o HTML é considerado o esqueleto de um site, pois esta define o conteúdo das páginas.

De seguida utilizamos o CSS - Cascading Style Sheets. Este é responsável pela apresentação da página Web. É excelente para "limpar" o HTML pois separa o layout (todo o aspecto visual do site no CSS) do conteúdo (código HTML e o próprio texto do site). Assim, a página Web criada fica em "camadas": uma é composta pelo conteúdo e o outra pelo layout (CSS). Deste modo, consegue-se economizar tempo quando queremos modificar algo na página Web e acaba por ser melhor indexado pelos browsers.

Para conseguir facilitar ainda mais a leitura do código, e para simplicar o máximo possivel a nossa folha de estilo (CSS), utilizámos a framework disponibilizada pela w3shools.

JavaScript é a linguagem de programação utilizado emHTML e nas internets.

Programação é um processo que conduz a partir de uma formulação original de um problema de computação para programas de computadores executáveis. A programação envolve atividades como a análise; desenvolvimento da compreensão; gerar algoritmos, entre outros. No fundo o objetivo da programação é encontrar uma sequência de instruções que irá executar uma tarefa especifica ou "resolver um determinado problema".

De um modo resumido programação torna o computador apto para fazer o que "nós" queremos.

Neste projeto foi também utilizado o JQuery. O objetivo desta ferramenta é tornar a utilização do JavaScript mais fácil no site em construção. Aliás o lema do jQuery é "Write less, do more."
